### 4.2 写入核心代码路径

```python
# enhanced_controller.py - write_memory()
def write_memory(self, text: str, source: str, metadata: dict, async_write: bool, role: str):
    # 1. 文本预处理
    text = self.preprocessor.preprocess(text)
    
    # 2. LLM结构化提取
    structured_summary = self.llm_extractor.extract(text)
    
    # 3. 质量评估
    quality = self.quality_evaluator.evaluate(text, {"structured_summary": ...})
    
    # 4. 向量嵌入
    vector = self.embedding_module.embed_text([text])[0]
    
    # 5. 去重检测
    dedup_result = self.deduplicator.check_duplicate(text, vector)
    
    # 6. 分层存储
    memory_id, level = self.layered_storage.store_memory(...)
    
    # 7. LanceDB存储
    self.store.add_memories([memory])
    
    # 8. 去重索引更新
    self.deduplicator.add(memory_id, text, vector)
    
    # 9. 冷热分层
    self.tiered_manager.put(tiered_item)
    
    # 10. 知识图谱更新
    self.memory_graph.add_node(...)
```

### 5.3 多路召回策略

```python
# unified_search.py - UnifiedSearchPipeline
class UnifiedSearchPipeline:
    def search(self, query, top_k, ...):
        # 并行执行多路召回
        results = []
        
        # 1. 热缓存搜索
        if self.config.enable_hot_cache:
            hot_results = self.hot_cache_searcher.search(query_vector, top_k)
            results.extend(hot_results)
        
        # 2. 向量搜索
        if self.config.enable_vector:
            vector_results = self.vector_searcher.search(query_vector, top_k)
            results.extend(vector_results)
        
        # 3. 关键词搜索
        if self.config.enable_keyword:
            keyword_results = self.keyword_searcher.search(query_text, top_k)
            results.extend(keyword_results)
        
        # 4. 图谱扩展
        if self.config.enable_graph and entities:
            graph_results = self.graph_searcher.search(entities, top_k)
            results.extend(graph_results)
        
        # 5. 情景记忆搜索
        if self.config.enable_episodic:
            episodic_results = self.episodic_searcher.search(query, top_k)
            results.extend(episodic_results)
        
        # 6. 核心规则搜索 (CORTEX_RULES.md)
        if self.config.enable_core_rules:
            core_rules_results = self.core_rules_searcher.search(query, top_k)
            results.extend(core_rules_results)
        
        # 7. 融合排序
        fused = self._fuse_results(results)
        
        # 8. 重排序
        if self.config.enable_rerank:
            reranked = self.reranker.rerank(query, fused)
        
        return reranked[:top_k]
```

## 总结

OpenClaw Cortex Memory 是一个完整的长期记忆系统，具有以下特点：

1. **分层架构**: TypeScript插件层 → Python API层 → 核心引擎层 → 存储层
2. **智能写入**: 预处理 → LLM提取 → 质量评估 → 三阶段去重 → 分层存储
3. **多路检索**: 热缓存 + 向量 + 关键词 + 图谱 + 情景记忆
4. **三维存储**: 分层存储(L0/L1/L2) × 冷热分层(HOT/WARM/COLD) × 向量存储
5. **自动触发**: 消息钩子自动存储和检索，会话结束自动生成事件
